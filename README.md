# hackathon-examples

## 一、运行步骤

**1、安装依赖**

> yarn

**2、打包文件**

> yarn build

**3、将打包后的 dist/logic 移动到 chrome_extension_dist/logic**

![move dir](https://public.feihan.cc/temp/demo_01.jpeg 'demo_01.jpeg')

**4、扩展程序加载 chrome_extension_dist 文件**

**5、预览 demo 效果**

打开[百度搜索页面](https://www.baidu.com/)即可看到插件注入的页面将原百度页面覆盖
